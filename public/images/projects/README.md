# Project Images

Place your project screenshot images in this directory:

- `bitcoin-miner.png` - Screenshot of your Bitcoin Miner app
- `sportstr.png` - Screenshot of Sportstr project
- `bitcoin-voice-price.png` - Screenshot of Bitcoin Voice Price app

## Recommended Image Specs

- **Format:** PNG or JPG
- **Dimensions:** 1200x675px (16:9 aspect ratio)
- **File size:** Under 500KB for optimal loading

## Tips for Good Screenshots

- Use browser screenshots at 1920x1080 resolution
- Crop to focus on the main interface
- Use dark mode if the app supports it
- Ensure text is readable

If you don't have screenshots yet, the site will show a placeholder with the project icon and title.
